PROMPT-01 · 整体风格定义
Design a cyberpunk UI with the following specifications: // ATMOSPHERE - Dark dystopian aesthetic, near-future neo-noir city feel - Heavy use of neon glow effects on interactive elements - CRT scanline overlay across entire viewport (2px lines, 5% opacity) - Moving scan beam animation from top to bottom, 4s loop - Background: near-black (#030712), not pure black // COLOR LANGUAGE - Primary accent: Electric Cyan (#00F5FF) — borders, glow, focus states - Secondary accent: Hot Magenta (#FF006E) — alerts, active, destructive - Tertiary: Void Purple (#B400FF) — decorative, hologram - Text: steel-tinted white (#C8D6EF), never pure white // TYPOGRAPHY - Display: Orbitron 900 weight, wide letter-spacing, ALL CAPS - UI text: Rajdhani 600, tracked uppercase for labels - Data/code: Share Tech Mono, matrix green (#39FF14)
